# OTT User Behavior Analysis (SQL Project)

## Project Overview
This project analyzes user behavior data from a simulated OTT streaming platform using MySQL. The goal is to extract meaningful business insights related to user engagement, subscription trends, and content performance.

## Tools Used
- MySQL
- SQL (JOIN, GROUP BY, Subqueries)
- Synthetic Dataset

## Database Structure
The database consists of the following tables:
- Users
- Subscriptions
- Content
- Watch_History

## Business Questions Solved
- Count on each subscription
- Which genre has the highest watch time?
- Average watch time per user
- Top 5 most watched contents
- Who are the most active users?
- Age group analysis
- Country-wise user distribution

## Key Insights
From this analysis, we observed that most users are on the free plan, drama-based content is the most popular, 
and young adults aged 18–35 form the largest user base. The platform can increase revenue by promoting premium plans, 
investing in popular genres, and targeting high-engagement regions with localized content.
